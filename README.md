# sokoban

Clojure app for playing sokoban.
Based on [RubyQuiz](http://www.rubyquiz.com/quiz5.html) and [The Caves of Clojure](http://stevelosh.com/blog/2012/07/caves-of-clojure-01/)

## Usage

h/j/k/k to move
r to restart
escape to exit
